# Security Policy

Prompt Firewall is a local redaction tool. It does not send text, prompts,
findings, or telemetry to any external service.

## Reporting Issues

Please open a GitHub issue for false positives, false negatives, or detector
bugs. Do not include real secrets, access tokens, customer data, or private
prompts in public issues.

If you need to demonstrate a detector problem, use fake values that match the
same shape as the sensitive value.

## Scope

Prompt Firewall is a practical guardrail, not a complete data-loss-prevention
system. Review sensitive material before sharing it with an AI service or third
party.
